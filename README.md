# Simple Todo App

This project is a simple todo app that allow the user to log task and due dates for each task.
## Functionality

The todo app was originally utilized procedural based coding. The forms have been refactored with form validation functionality, unique ID's to each todo item, and all todo generation is through the object class.

## Technology

JavaScript(ES6+), HTML/CSS, UUID Library(for generating unique IDs), DOM Manipulation, and OOP (object-Oriented Programming).
